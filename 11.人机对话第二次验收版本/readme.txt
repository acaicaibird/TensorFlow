对话系统第二版改进点:
	1.采用flask框架，web服务更稳定
	2.满足陈勇要求:每次启动服务前不训练，根据json命令才训练，0条数据服务可启动
	3.满足陈总要求:一份代码对外提供所有服务，无需拷贝多个文件
	4.所有功能均支持命令行参数配置
	5.优化剩余时间计算方式，避免之前数据量太多时，时间没有更新，测试人员以为训练没有开始
	6.优化代码结构，节省内存，调参和对外服务均为一份代码，提高可维护性
	7.训练阶段可对话
	
使用方法:
	cd server
	python3 run_server.py -u 2 -m ../data/runs/test1 -p 1111

测试方法:
	cd server
	python3 http_test.py
	输入:l 代表查询训练进度
	输入:t 代表重新训练
	输入:c 代表对话
